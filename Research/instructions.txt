System Requirements:

*WAMP Server
*Web Browser(Any)

----------

How to copy the system file(WAMP Server already installed):

*Go to wamp>www folder and copy the folder(onlinebiddingsystem)

----------

How to import the database tables:

*create a database named biddingsystemdb in phpmyadmin
*click import and then browse wamp>www>biddingsystem>database select the biddingsystemdb.sql
